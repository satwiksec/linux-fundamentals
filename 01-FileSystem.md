# Files and Directories

## touch

Creates a new empty file.

Example:

```bash
touch notes.txt
```


## ls

Lists the contents of the current directory.

Example:

```bash
ls
```


## ls -l

Displays a long listing format with additional details such as:

- Permissions
- Owner
- Group
- File size
- Last modification date

Example:

```bash
ls -l
```


## ls -a

Displays all files, including hidden files.

In Linux, files beginning with a dot (`.`) are hidden.

Example:

```bash
ls -a
```


## pwd

Stands for **Print Working Directory**.

Shows the absolute path of the current directory.

Example:

```bash
pwd
```



## cd

Stands for **Change Directory**.

Used to move between directories.

Example:

```bash
cd research
cd ..
```



## cat

Displays the contents of a file without opening a text editor.

Example:

```bash
cat notes.txt
```



## mkdir

Stands for **Make Directory**.

Creates a new directory (folder).

Example:

```bash
mkdir research
```

This creates a directory named `research`.




## ls -la

A combination of:

```bash
ls -l
ls -a
```

Displays detailed information about all files, including hidden files.

Example:

```bash
ls -la
```
